# Guessing Number Game

A Java console game where the player tries to guess a randomly generated number within a limited number of attempts.

## How It Works

The program generates a random number between 1 and 100. The player picks a difficulty level which determines how many attempts they get — Easy (10), Medium (5), or Hard (3). After each guess, the program tells the player whether the target number is bigger or smaller until they either find it or run out of attempts.

## How to Run

1. Open the project in your Java IDE
2. Run "GuessingNumberGame.java"
3. Select a difficulty level (1, 2, or 3)
4. Start guessing

## Background

Built to practice core Java concepts including loops, conditionals, user input handling, and random number generation.
